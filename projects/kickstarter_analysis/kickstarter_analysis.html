Kickstarter Project Analysis

In this project, we worked on a dataset with information about Kickstarter projects from 2009 to 2019.

This was one of the projects we did during the neuefische Data Science Bootcamp.

The GitHub repository for this project can be found here,
as well as the presentation of the results.

Project Background

  • The goal of this data science project was to identify what makes a Kickstarter project successful.
  • With this knowledge, creators could be helped to successfully fund their new projects, which would increase Kickstarter's (our stakeholder) revenue.
  • In addition, this would also lead to higher satisfaction among Kickstarter's customers (the creators), as their projects would have higher success rates.
